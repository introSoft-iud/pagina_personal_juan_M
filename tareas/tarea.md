## Implementando la tortuga

Para hacer mover la tortuga hacia adelante, hacemos lo siguiente:

```python
import turtle

# Inicializar la tortuga
mi_tortuga = turtle.Turtle()

# Mover hacia adelante 100 pasos
mi_tortuga.forward(100)
```
Así es como sumamos los números de una lista:

```python
# Sum all the numbers in the list xs
def sum_all(xs):
    total = 0
    for x in xs:
        total = total + x
    return total
```
